# 米豆 — 陪你一起照顧寶貝的育兒資訊站

[mitou.app](https://mitou.app) 是聚焦台灣爸媽的育兒資訊平台，從新手爸媽到學齡前，把育兒生活中最需要的資訊整理成一個查得到、看得懂、決定得了的入口。我們相信爸媽不該在最忙亂的時候還要四處比對、來回切換，所以把醫療、托育、親子場域、寶寶用品與育兒創作者放在同一個畫面，陪你照顧好寶貝的每一個成長瞬間。

## 核心價值

- **找醫療更有底氣**：把就醫時需要的資訊整理在同一個入口，需要的時候能馬上比對選擇。
- **托育決定不焦慮**：把托嬰與幼教資訊集中呈現，就近比較不用切換多個網站。
- **假日不再卡關**：把親子友善的場域整理成清單，帶小孩出門前就先做好功課。
- **採購少走冤枉路**：把寶寶常用品項的重點整理在一頁，減少新手爸媽的選擇疲勞。
- **找到值得追蹤的人**：收錄育兒創作者頁面，讓有品質的內容更容易被看見。

## 使用方式

米豆網站公開使用，無需註冊即可查詢全部內容。
分類與地區篩選直接呈現在頁面上，符合爸媽零碎時間快速比對的習慣。
